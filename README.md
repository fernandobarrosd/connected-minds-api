# Connected Minds API

## Funcionalidades

### Autenticação

- O usuário poder se autenticar
- O usuário poder criar uma conta

### Posts

- Criar um post em uma comunidade ou grupo
- Deletar um post
- Editar um post
- Curtir um post
- Comentar em um post
- Tirar a curtida do post

### Comentários

- Comentar em um post
- Remover o comentário do post
- Editar o comentário de um post
- Curtir um comentário
- Remover a curtida do comentário

### Grupos

- Criar um grupo em uma comunidade
- Deletar um grupo
- Editar um grupo
- Convidar membros pra um grupo
- Entrar em um grupo
- Sair de um grupo

### Comunidades

- Criar uma comunidade
- Editar a comunidade
- Deletar uma comunidade
- Convidar membros pra comunidade
- Entrar em uma comunidade
- Sair de uma comunidade


### Favoritos
- Favoritar um post, comunidade ou grupo
- Remover um post, comunidade ou grupo dos favoritos

### Buscar

- Buscar um grupo ou comunidade pela TAG
- Buscar um usuário
- Buscar um grupo ou comunidade
